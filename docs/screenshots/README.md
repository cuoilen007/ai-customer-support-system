# Screenshot Guide

Recommended screenshots for this repository:

- `chat-page.png`: main support chat with realistic seeded data
- `ai-review.png`: grouped low-confidence reviews with correction workflow
- `dashboard.png`: analytics dashboard with training history and review charts
- `knowledge-base.png`: documents, products, or policies management view

Capture tips:

- Use real data instead of empty states
- Keep browser zoom at 100%
- Prefer a clean desktop viewport
- Show the most information-rich state of each page
